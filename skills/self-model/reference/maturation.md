# Knowledge Maturation Model

```
Gas    — Human's ambiguous intent
  | Translation swamp
Liquid — Agent society (Quest structuring)
  | Quality gate
Solid  — Heritage (immutable decisions + failure patterns)
```

## Self-sufficiency Stages

| Stage | Name | Description |
|-------|------|-------------|
| A | Dependent | Agent requires human guidance for most decisions |
| B | Assisted | Agent operates independently 30%+ of the time |
| C | Balanced | Agent self-sufficient 100%, human reviews only |
| D | Growing | Agent improves the system beyond initial design |

## Transition Signals

- A->B: Sprint completes without intervention 3+ times
- B->C: Gate chain passes first-try 5+ consecutive sprints
- C->D: Agent proposes and implements system improvements autonomously

## Cognitive Spiral

Idea (quantity) -> Insight (1-sentence compression) -> Design (minimum verifiable unit) -> Evolution (real-world feedback)
