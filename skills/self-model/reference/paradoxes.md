# Design Paradoxes (5 Trade-offs)

1. **Observation vs Intervention** (Seventh Day)
   Higher maturity phase -> more observation, less intervention.
   Premature intervention corrupts the feedback loop.

2. **Freedom vs Exploitation** (Forbidden Fruit)
   Grazing within fences. 4 anti-gaming types:
   metric manipulation, scope inflation, heritage padding, gate bypass.

3. **Efficiency vs Resilience** (Manna)
   40% reserve determines crisis survival.
   Over-optimization = brittle. Single Quest limit: 30% of Pool.

4. **Speed vs Quality** (Babel)
   Heritage contamination is irreversible.
   Self-scoring bias exists — agents rate own work too highly.
   External verification required for quality claims.

5. **Human Bottleneck** (Moses)
   Small certain Quests -> rapid Trust accumulation.
   Constitution governance as intentional safety mechanism, not bug.
